## 🚀 Usage

Minimal example:

```yaml
jobs:
  asset-analysis-test:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout repository
        uses: actions/checkout@v6

      - name: Run MCIX Asset-Analysis Test
        id: asset-analysis-test
        uses: ${{ github.repository }}/asset-analysis/test@v1
        with:
          api-key: <required>
          url: <required>
          user: <required>
          path: <required>
          rules: <required>
          # report: reports/asset-analysis-report.xml
          # project: <optional>
          # project-id: <optional>
          # include-tags: false
          # exclude-tags: <optional>
          # test-suite: mcix tests
          # ignore-test-failures: false
          # additional-args: <optional>
```
